# Calculadora - Projeto de Portfólio

Este é um projeto simples de uma calculadora desenvolvida como parte do meu portfólio. A calculadora é uma aplicação web que permite realizar operações matemáticas básicas.

## Tecnologias Utilizadas

**Front-End:**

- HTML
- CSS
- JavaScript

**Back-End (Opcional):**

- Python

## Funcionalidades

- Adição, subtração, multiplicação e divisão.
- Exibição dinâmica dos números e operadores no visor.
- Design responsivo para melhor experiência em diferentes dispositivos.
